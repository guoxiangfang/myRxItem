告警汇聚

- OneAlert 内置12+监控工具/平台告警集成

- 更提供强大的 REST API 方式接入告警

- 所有成员，均可在 Cloud Alert 认领、关闭告警、记录解决方案，解决方案可在下次相似问题发生后自动推荐给您

降噪

- 告警发到 Cloud Alert 后，系统会自动压缩重复数据，平均降噪60%，通过算法降噪、熵计算，直观查看重要告警

- 进一步提升降噪比高达98%。Cloud Alert 降噪功能无需设定规则，自动开启，闷声干大事

聚类

- 将一段时间内的告警，通过无监督机器学习自动聚类告警形成场景，聚焦处理问题而非单个告警

- Cloud Alert算法集群引擎结合有监督和无监督AI能力，为您提供自定义标签入口，方便您从业务服务顶层视角直接查看管理的场景和告警

分派必达

- 基于应用的告警分派，配合基于业务的场景分派，2套机制双重保障分派到责任人，更有排班管理助力

6 种通知方式，让你不错过重要告警。ChatOps，如钉钉、微信、APP还可认领、关闭告警，双向沟通

根因定位

- 在场景中，同样结合有监督和无监督AI能力，快速标注根因or非根因，系统根据场景自动推荐可能根因

- 随着告警规模的增加，系统推荐告警根因将更加准确

知识复用

- 通过算法，根据相似度大小，自动推荐历史相似场景/告警，关联历史解决方案，帮助更快解决问题

- 每个场景的解决方案，将成为您团队知识库的养分，持续帮助您提升团队的知识
